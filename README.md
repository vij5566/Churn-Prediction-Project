# Churn-Prediction-using-ANN-classification
# 📊 Customer Churn & Salary Prediction Dashboard

This project predicts:
- Customer churn (classification)
- Customer salary (regression)

## Features
- ANN-based models using TensorFlow
- Streamlit interactive dashboard
- Real-time predictions
- Clean UI with business insights

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Streamlit
- Pandas, NumPy

## ▶️ Run Locally

```bash
pip install -r requirements.txt
